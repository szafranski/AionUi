## Phase 2: 规划故事（3D 专属内容）

```
读 reference/decision-rules.md
        │
        ▼
┌─────────────────────┐
│ 写 brief.md           │
│                       │
│ ① 标准内容：           │
│   • 主题/受众/目的     │
│   • SCQA 叙事结构      │
│   • 逐页大纲           │
│                       │
│ ② Morph Pair Scene 表  │
│   • !! 形状命名        │
│   • 入场/退场时机       │
│                       │
│ ③ 模型编排表 (3D新增)   │
│   见下表 ↓              │
└──────────┬────────────┘
           │
           ▼

┌──────┬───────────┬──────────┬──────┬──────┬──────────┐
│ 页    │ 镜头类型     │ 布局模式   │ roty │ rotx │ 模型尺寸  │
├──────┼───────────┼──────────┼──────┼──────┼──────────┤
│ 1    │ 3/4美拍     │ C 居中    │ 30   │ 8    │ 20×17cm  │
│ 2    │ 特写       │ A 模型右   │ 0    │ 5    │ 26×22cm  │
│ 3    │ 侧面       │ B 模型左   │ 80   │ 0    │ 15×14cm  │
│ 4    │ 鸟瞰       │ D 模型小   │ 120  │ 35   │ 16×14cm  │
│ 5    │ 仰拍       │ A 模型右   │ 170  │ -20  │ 18×16cm  │
│ 6    │ 过肩       │ D 模型小   │ 220  │ 10   │ 11×10cm  │
│ 7    │ 全景       │ C 居中    │ 320  │ 5    │ 20×17cm  │
└──────┴───────────┴──────────┴──────┴──────┴──────────┘

检查规则：
  □ 至少用了 3 种不同镜头类型？
  □ 相邻 roty 差值 30°-90°？
  □ 没有连续 3 页同尺寸？
  □ 布局模式有交替变化？
```

## Phase 3: 设计生成（3D 构建流程）

```
┌─────────────────────────────────────┐
│ Step 1: 创建所有 slide                 │
│                                       │
│ for i in 1..N:                        │
│   officecli add / --type slide        │
│     --prop background=COLOR           │
│     --prop transition=morph           │
└──────────────────┬──────────────────┘
                   │
                   ▼
┌─────────────────────────────────────┐
│ Step 2: 加场景演员 (slide 1)           │
│                                       │
│ add !!scene-ring  (装饰圆环)           │
│ add !!scene-dot   (装饰圆点)           │
│ add !!scene-line  (装饰线条)           │
│                                       │
│ ⚠️ 场景演员可以 clone                  │
│ for i in 2..N:                        │
│   helper("clone", OUTPUT, 1, i)       │
└──────────────────┬──────────────────┘
                   │
                   ▼
┌─────────────────────────────────────┐
│ Step 3: 每页独立 add 3D 模型           │
│                                       │
│ ⚠️ 绝对不能 clone 3D 模型！            │
│                                       │
│ slide 1: add 3dmodel                  │
│   name=!!model-hero                   │
│   roty=30, x=7cm, w=20cm             │
│                                       │
│ slide 2: add 3dmodel  ← 重新 add      │
│   name=!!model-hero   ← 同名！        │
│   roty=0, x=4cm, w=26cm              │
│                                       │
│ slide 3: add 3dmodel  ← 重新 add      │
│   name=!!model-hero   ← 同名！        │
│   roty=80, x=1cm, w=15cm             │
│   ...                                 │
└──────────────────┬──────────────────┘
                   │
                   ▼
┌─────────────────────────────────────┐
│ Step 4: 每页加内容，ghost 上一页内容     │
│                                       │
│ slide 1:                              │
│   add #s1-title                       │
│   add #s1-body                        │
│                                       │
│ slide 2:                              │
│   helper("ghost", OUTPUT, 2, ...)     │
│   ← ghost 掉 #s1-title, #s1-body     │
│   add #s2-title                       │
│   add #s2-body                        │
│                                       │
│ slide 3:                              │
│   helper("ghost", OUTPUT, 3, ...)     │
│   add #s3-title                       │
│   ...                                 │
│                                       │
│ 同时调整 !!scene-* 位置制造动画         │
└──────────────────┬──────────────────┘
                   │
                   ▼
┌─────────────────────────────────────┐
│ Step 5: 收尾                          │
│                                       │
│ officecli close OUTPUT                │
│ helper("final-check", OUTPUT)         │
└─────────────────────────────────────┘
```

## Phase 4: 验证交付

```
┌───────────────────────────────────┐
│ 4A: CLI 验证                        │
│                                     │
│ helper("final-check", OUTPUT)       │
│ officecli validate deck.pptx        │
│ officecli view deck.pptx outline    │
│                                     │
│ 检查：                               │
│ □ 每页 transition=morph?            │
│ □ #sN- 内容全部 ghost 到 x=36cm?    │
│ □ officecli validate 通过?          │
└────────────────┬──────────────────┘
                 │
                 ▼
┌───────────────────────────────────┐
│ 4B: 3D 专属检查                     │
│                                     │
│ □ 每页有且只有 1 个 model3d?         │
│ □ 所有模型 name 相同?               │
│ □ 模型框在画面范围内?               │
│ □ 模型和内容文字不重叠?             │
└────────────────┬──────────────────┘
                 │
                 ▼
┌───────────────────────────────────┐
│ 4C: 截图目视验证                     │
│                                     │
│ officecli view deck.pptx svg        │
│   --output-dir screenshots/         │
│                                     │
│ 逐页检查：                           │
│ □ 模型可见且大小符合规划?            │
│ □ 前一节 !! shapes 已 ghost?        │
│ □ 镜头类型变化有节奏?               │
│ □ 模型角度配合页面内容?             │
└────────────────┬──────────────────┘
                 │
                 ▼
         交付给用户
         "打开后按 F5 放映"
```

## 镜头语言速查

```
               rotx
               +40° (鸟瞰)
                │
                │   Bird's eye
                │   俯视结构
                │
    ────────────┼──────────── roty
   270°         │ 0°       90°       180°
   (左侧)       │(正面)   (右侧)    (背面)
                │
                │   Low angle
                │   仰拍气势
                │
               -25° (仰拍)

模型大小 = 镜头距离：
  28cm = 特写 (模型超出画面)
  20cm = 全景 (完整展示)
  14cm = 中景 (平衡文字)
  10cm = 远景 (模型做点缀)
```
